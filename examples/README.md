# Examples

## General Tests

```
$ test/run.sh
```

- Requirement: vagrant or boot2docker for Mac OS X, docker for Linux

## Build WordPress

```
$ wordpress/run.sh
$ open http://localhost:8000/
```

- Requirement: vagrant or boot2docker for Mac OS X, docker for Linux

Cf.) https://github.com/docker/fig/blob/master/docs/wordpress.md
